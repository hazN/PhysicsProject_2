PRESS ESCAPE UNLOCK/LOCK THE MOUSE

Mode is automatically set to PHYSICS, if you'd like to use the free-move camera
then click camera at the top left, then press ESC to lock the mouse.

Implemented PhysX engine.

Controls(PHYSICS):

WASD -> Ball Movement
UP/DOWN ARROW -> Zoom In/Out
LEFT/RIGHT ARROW -> Rotate Camera
NUM_1 -> Spawn Random Ball
NUM_0 -> Reset Scene

Controls(CAMERA):

WASD -> Camera Movement
Q/E -> Up/Down
ARROW KEYS -> Ball Movement
NUM_1 -> Spawn Random Ball
NUM_0 -> Reset Scene
